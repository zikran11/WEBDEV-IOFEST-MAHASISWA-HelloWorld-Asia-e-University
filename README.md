# ♻️ Smart Waste - AI Powered Waste Management Platform  

Smart Waste adalah platform web inovatif yang menggabungkan Kecerdasan Buatan (AI) dengan konsep circular economy untuk membantu pengguna mengidentifikasi jenis limbah, memahami dampak lingkungannya, serta mengubah sampah menjadi nilai ekonomi melalui edukasi dan teknologi digital.  

**Vercel Deployment • Next.js • Tailwind CSS**

## ✨ Fitur Utama  
- 🔍 **AI Waste Scanner**: Identifikasi jenis limbah (Organik, Anorganik, B3) secara cepat melalui foto menggunakan teknologi AI Vision.  
- 💰 **Waste Value Insight**: Menyediakan informasi potensi nilai ekonomi dari limbah yang diunggah, termasuk peluang daur ulang dan pemanfaatan kembali.  
- 🌱 **Eco Impact Analysis**: Menampilkan dampak lingkungan dari limbah tertentu serta solusi pengelolaan berkelanjutan.  
- 📚 Artikel & Blog Edukasi: Pusat informasi yang menyediakan artikel, blog, serta wawasan seputar pengelolaan sampah, circular economy, sustainability, dan gaya hidup ramah lingkungan.
- 📱 **Mobile First Design**: Desain responsif dan modern yang optimal untuk desktop maupun smartphone.  
- 🌙 **Modern Green Interface**: Tampilan futuristik dengan nuansa hijau sebagai simbol teknologi dan lingkungan.  
- 🔄 **User Dashboard**: Dashboard personal untuk melihat riwayat scan, aktivitas pengguna, dan insight pengelolaan limbah.  
- 🔐 **Secure Authentication**: Sistem Login & Sign Up untuk akses yang aman dan personal.  

## 🚀 Tech Stack  
- **Frontend:** Next.js  
- **Styling:** Tailwind CSS  
- **AI Engine:** Google Gemini AI 
- **Database:** Firebase 
- **Deployment:** Vercel  

## 🛠️ Cara Menjalankan Lokal  

### 1. Clone Repository  
```bash
git clone https://github.com/zikran11/WEBDEV-IOFEST-MAHASISWA-HelloWorld-Asia-e-University.git
cd WEBDEV-IOFEST-MAHASISWA-HelloWorld-Asia-e-University
```
### 2. Install Dependencies

```bash id="h1w8kv"
npm install
```

### 3. Konfigurasi Environment Variable  

```bash
GEMINI_API_KEY=masukkan_api_key_anda_disini

NEXT_PUBLIC_FIREBASE_API_KEY=masukkan_firebase_api_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=project-id.firebaseapp.com
NEXT_PUBLIC_FIREBASE_PROJECT_ID=project-id
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=project-id.appspot.com
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=masukkan_sender_id
NEXT_PUBLIC_FIREBASE_APP_ID=masukkan_app_id

NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=masukkan_cloud_name
NEXT_PUBLIC_CLOUDINARY_UPLOAD_PRESET=masukkan_upload_preset
```

### 4. Jalankan Development Server

```bash id="yeq4go"
npm run dev
```

### 5. Buka di Browser
